Свойства CSS

CSS (Cascading Style Sheets) - это язык стилей, который позволяет задавать внешний вид элементов на веб-странице. С помощью CSS можно изменять цвет, размер, шрифт, фон, границы, отступы и другие свойства элементов.

Для применения стилей к элементам используются селекторы. Селекторы могут быть разных типов, например:

  • Селекторы тегов - применяют стили ко всем элементам с заданным тегом.
  • Селекторы классов - применяют стили к элементам с заданным атрибутом class.
  • Селекторы идентификаторов - применяют стили к элементу с заданным атрибутом id.
  • Селекторы атрибутов - применяют стили к элементам с заданным атрибутом или его значением.
  • Селекторы псевдоклассов - применяют стили к элементам в зависимости от их состояния или положения.
  • Селекторы псевдоэлементов - применяют стили к частям элементов.
  • Комбинированные селекторы - применяют стили к элементам, которые удовлетворяют нескольким условиям одновременно.

Для задания стилей используются свойства и их значения. Свойства могут быть разных групп, например:

  • Свойства текста - позволяют изменять цвет, шрифт, размер, выравнивание, интервал и другие характеристики текста.
  • Свойства фона - позволяют изменять цвет, изображение, положение и повторение фона элемента.
  • Свойства границы - позволяют изменять толщину, стиль, цвет и радиус границы элемента.
  • Свойства отступов - позволяют изменять расстояние между границей элемента и его содержимым (padding) или между границей элемента и другими элементами (margin).
  • Свойства размеров - позволяют изменять ширину и высоту элемента.
  • Свойства позиционирования - позволяют изменять способ расположения элемента на странице.
